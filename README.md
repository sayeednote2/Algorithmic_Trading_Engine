# 🚀 Algorithmic Trading Engine

## 📌 Overview

This project is a **real-time algorithmic trading engine** built using Python.
It focuses on **Renko chart logic**, **custom indicators**, and **live market data integration**.

The system is designed for **speed, simplicity, and modular trading strategy development**.

Full End to End Trading Engine:
 1. 1-second interval Charting.
 2. Custom proprietary indicators.
 3. Signal Generation.
 4. Ultra Fast High Frequency(milli second level) Trade Execution.

---

## ⚙️ Features

* 📊 Real-time Renko chart generation
* 📈 Custom indicators (RSI, Variance, Volume Profile)
* 🔌 Broker integration (Fyers API)
* ⚡ Fast data handling using Redis
* 📡 WebSocket-based live updates
* 🖥️ Interactive dashboard interface

---

## 🏗️ Architecture

* **Backend**: Python
* **Data Streaming**: WebSocket
* **Cache Layer**: Redis
* **Broker API**: Fyers
* **Frontend**: Flask
